****** How to compile and run *******

This code is created and executed on IntelliJ Idea, and it has compiled all the latest modifications into the folder name ‘out’.
You can run compile and run only the ‘Runner.java’ in the CMD or in any smart IDE, because ‘Runner.java’ contains the main method to execute the program.
You can run the file in CMD by following these steps:
	1) type 'javac Runner.java Node.java SetClass.java'
	2) to run type 'java Runner'
For accessing and running the test cases file you have to change the path inside the program in Runner.java file by changing the line 7 path to the path of your testcase file.
